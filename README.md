BlitzSG
Blitz Survival Games minigame plugin for Paper 1.20.4 made in blitz time!

Multi-arena support with independent game instances
8 kits with passive and active abilities
Tiered chest loot system with configurable loot tables
Game state machine: Waiting → Countdown → Grace Period → Active → Deathmatch → Ended
Closing WorldBorder deathmatch, spectator system, lifetime stats, public API
